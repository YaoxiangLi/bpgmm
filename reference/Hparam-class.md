# Hyperparameter set for the Bayesian PGMM sampler.

Hyperparameter set for the Bayesian PGMM sampler.

## Slots

- `alpha1`:

  First Dirichlet prior parameter for component weights.

- `alpha2`:

  Second Dirichlet prior parameter for component weights.

- `delta`:

  Shape parameter used in prior updates.

- `ggamma`:

  Prior rate parameter used in covariance updates.

- `bbeta`:

  Prior scale parameter used in covariance updates.
