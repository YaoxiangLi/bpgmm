# CalculateProposalLambda

CalculateProposalLambda

## Usage

``` r
CalculateProposalLambda(hparam, thetaYList, CxyList, constraint, m, p, qVec)
```

## Arguments

- hparam:

  hparam

- thetaYList:

  thetaYList

- CxyList:

  CxyList

- constraint:

  constraint

- m:

  the number of clusters

- p:

  the number of features

- qVec:

  the vector of the number of factors in each clusters
