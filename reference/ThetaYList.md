# ThetaYList-class

Parameter set for sampled PGMM component parameters.

## Slots

- `tao`:

  Numeric vector of component mixing weights.

- `psy`:

  List of diagonal noise covariance matrices.

- `M`:

  List of component mean vectors.

- `lambda`:

  List of component factor loading matrices.

- `Y`:

  List of latent factor score matrices.
