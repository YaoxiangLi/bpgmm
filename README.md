# bpgmm

[![CRAN status](https://www.r-pkg.org/badges/version/bpgmm)](https://cran.r-project.org/package=bpgmm)

Bayesian Model Selection Approach for Parsimonious Gaussian Mixture Models.

Model-based clustering using Bayesian parsimonious Gaussian mixture models. MCMC
(Markov chain Monte Carlo) is used for parameter estimation. RJMCMC
(Reversible-jump Markov chain Monte Carlo) is used for model selection.

```r
install.packages("bpgmm")
```
